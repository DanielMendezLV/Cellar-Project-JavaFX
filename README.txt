Este es un proyecto de unidad realizado en javaFx que consta de 3 visores y mantiene un control de de datos

Los tres modulos visibles son los siguientes: CRUD, ModuloPrestamista, ModuloVisor
El modulo CRUD contiene: Eliminar, Leer, Update, Delete de los siguientes campos: Categoria, Profesores, Usuario, Tipo Y herramientas

El programa esta hecho para mantener un control de una bodega de la forma mas precisa posible
